API Info:

The API is based on the nzedb/newznab API, it's similar but there are some differences.

Things not included (because they are incompatible with our site):

Registration: ?t=register
User: ?t=user
Cart: ie: ?t=cartadd / ?t=cartdelete
Searching TV, Movies, Music, Books: ?t=tvsearch / ?t=movie / etc..
Comments: ?t=comments
Extended attributes: ?t=attrs / ?extended=1

Things that are the same.

Capabilities: http://{$serveraddress}/api.php?t=caps | Displays the capabilities of the API.
Get: http://{$serveraddress}/api.php?t=get&id=SHA1HASH | Gets an NZB file for the collection using its hash, optionally pass the groupid to speed it up (&gid=2)
Details: http://{$serveraddress}/api.php?t=details&id=SHA1HASH | Loads the page for the collection, optionally pass the groupid to speed it up (&gid=2)
Output format: &o=xml | &o=json returns XML or JSON, if you don't specify it defaults to xml.

Things that are changed.

Search: http://{$serveraddress}/api.php?t=search&q=linux | The &sort=size_asc will be added, it's not added yet.
minsize | maxsize | group are identical : http://{$serveraddress}/api.php?t=search&q=linux&group=a.b.multimedia&minsize=0&maxsize=734003200
There is no categories in this site so it's not present.
